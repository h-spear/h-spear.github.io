이 문제는 특정 알고리즘이나 자료구조를 요구하지 않는 단순 **구현(Implementation) 및 시뮬레이션(Simulation)** 문제입니다.  
제약 조건이 작기 때문에, 문제에서 요구하는 대로 구현하면 됩니다. 핵심은 길고 복잡한 지문의 내용을 실수 없이 코드로 작성하는 것입니다.  
기존에 접해왔던 코딩 테스트 구현 문제와 다르게 객체지향(OOP) 개념을 활용하면 더 편하게 해결할 수 있는 고난이도 구현 문제였습니다.

## **설계와 구현 전략**
---

### **객체지향 설계**

게임의 핵심 요소인 `Player`, `Monster`, `Item`, `Map`을 각각의 클래스로 분리하여, 하나의 클래스가 **단 하나의 책임**만 갖도록 역할을 부여했습니다.  
전체적인 설계 구조는 다음과 같습니다.  

![BOJ_17081_Simple-Diagram](/assets/img/post/boj/17081/diagram.png) 

각 클래스의 구체적인 역할은 다음과 같습니다.  
	
- **Table**
	- 게임의 상호작용을 조정하는 **중재자(Mediator) 역할**
	- 문제의 입력 데이터 파싱 및 객체 생성 책임

- **<span style="background-color:yellow">Map</span>**
	- 맵의 크기와 **주인공, 몬스터, 아이템의 위치 정보** 관리
	- `isWall`, `getItem` 등 **맵의 정보를 외부에 제공**
	- `removeMonster`, `removeItem` 등 **내부 상태 변경 책임**

- **<span style="background-color:yellow">Player</span>**
	- 게임의 핵심 요소
	- 플레이어의 위치, 스탯, 장비 등 모든 상태를 관리하는 **데이터 모델 역할**
	- 전투(`battleWith`), 아이템 장착(`equipItem`) 등 다른 객체와 상호작용하는 핵심 로직 담당

- **Monster**
	- `Player`와 달리 복잡한 로직을 수행하지는 않음
	- 몬스터의 이름, 스탯 등 문제에서 주어진 **데이터를 보관하는 역할에 집중**
	- 보스 몬스터 여부는 `isBoss` 필드로 구분

- **Item**
	- 아이템의 다양성을 표현하기 위해 **상속(Inheritance)**을 활용한 **데이터 모델**
	- `무기(Weapon)`, `갑옷(Armor)`, `장신구(Accessory)`가 `Item` 추상 클래스를 상속받는 구조
	- `장신구`의 여러 가지 효과는 문자열 대신 **`Enum`**으로 정의하여 코드의 안정성과 가독성을 높임


### **구현 전략1: 턴 진행**
게임의 전체 흐름은 `Table` 클래스의 `simul()` 메서드가 수행하며, 그 안의 `for` 루프가 플레이어의 이동 커맨드를 하나씩 순회하면서 `turn()` 메서드를 호출합니다.  
```
For each character 'direction' in 캐릭터_이동_정보_문자열:
	턴 증가
	turn(direction)
End For
```
`turn()` 메서드는 `Player`와 `Map` 객체를 통해 그 턴에 일어나는 이벤트를 처리합니다.  
주인공의 다음 이동 위치를 계산하고, 그리드의 종류에 따라 전투, 아이템 획득, 가시 함정 등의 이벤트를 처리한 후 최종적으로 주인공의 위치를 업데이트합니다.  

`battleWith()`, `equipItem()`, `stepSpikeTrap()` 등 각각 이벤트에 대한 처리 로직은 코드에서 확인하실 수 있습니다.

### **구현 전략2: Exception을 활용한 게임 종료 처리**

![BOJ_17081_Exception-Handling](/assets/img/post/boj/17081/exception-handling.png) 

게임이 종료되는 조건(주인공의 죽음, 보스 몬스터 처치)은 상태 변수가 아닌 `GameEndException`으로 처리했습니다.  

`boolean` 플래그와 같은 상태 변수로 게임 종료 상태를 관리하면, 그 상태를 매 메서드마다 확인하고 전파해야 해서 번거로워집니다.  
이를 `Exception`을 사용하고 최상단 `simul()`메서드에서 `try-catch` 블록으로 감싸면, 게임 종료 조건이 충족되는 즉시 **모든 중간 과정을 건너뛰고 catch 블록으로 제어 흐름을 점프**시킬 수 있습니다.

이 덕분에 중간 메서드들이 게임 종료 여부를 신경 쓸 필요가 없어 코드가 간결해집니다.  

### **구현 전략3: toString()을 활용한 화면 출력**
문제에서 요구하는 최종 출력은 각 객체의 `toString()` 메서드를 **오버라이드(Override)**하는 방식으로 구현했습니다. `Player`는 자신의 스탯 정보, `Map`은 그리드의 현재 상태를 문자열로 반환하도록 책임을 부여했고, `Table` 클래스에서 이 결과들을 조합하여 최종 출력 형태를 완성합니다.  
```java
// Table 클래스의 toString (전체 출력 조합)
@Override
public String toString() {
	return map.toString() + "\n"
		+ String.format(Messages.TURN_PRINT_FORMAT, passedTurns) + "\n"
		+ player.toString() + "\n"
		+ reason;
}
```

## **풀이**

---

전체 코드는 다음과 같습니다.  

### **Java 코드**
```java
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class Main {

	public static void main(String[] args) throws Exception {
		Table table = new Table();
		table.input();
		System.out.println(table.simul());
	}
}

class Table {

	private int passedTurns;
	private Map map;
	private Player player;
	private String reason;

	public String simul() {
		passedTurns = 0;
		reason = Messages.CONTINUE;
		try {
			for (char direction: player.command) {
				++passedTurns;
				turn(direction);
			}
		} catch (GameEndException e) {
			reason = e.getMessage();
		}
		return toString();
	}

	private void turn(char direction) {
		Position nextPosition = player.nextPosition(direction);
		if (!map.isValid(nextPosition) || map.isWall(nextPosition)) {
			nextPosition = player.position;
		}
		char next = map.getValue(nextPosition);
		boolean catchBoss = false;
		if (next == Map.SPIKE_TRAP) {
			player.stepSpikeTrap();
			if (player.activateReincarnation) {
				player.reincarnation();
				return;
			}
		} else if (next == Map.MONSTER || next == Map.BOSS_MONSTER) {
			Monster monster = map.getMonster(nextPosition);
			player.battleWith(monster);
			if (player.activateReincarnation) {
				player.reincarnation();
				return;
			}
			map.removeMonster(nextPosition);
			if (monster.isBoss) {
				catchBoss = true;
			}
		} else if (next == Map.ITEM_BOX) {
			Item item = map.getItem(nextPosition);
			player.equipItem(item);
			map.removeItem(nextPosition);
		}
		player.move(nextPosition);
		if (catchBoss) {
			throw new GameEndException(Messages.WIN);
		}
	}

	public void input() throws IOException {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st;

		st = new StringTokenizer(br.readLine());
		int N = Integer.parseInt(st.nextToken());
		int M = Integer.parseInt(st.nextToken());
		char[][] grid = new char[N][M];

		int K = 0;
		int L = 0;
		Position playerPosition = null;
		for (int i = 0; i < N; ++i) {
			String line = br.readLine();
			for (int j = 0; j < M; ++j) {
				grid[i][j] = line.charAt(j);
				if (grid[i][j] == Map.MONSTER || grid[i][j] == Map.BOSS_MONSTER) {
					++K;
				} else if (grid[i][j] == Map.ITEM_BOX) {
					++L;
				} else if (grid[i][j] == Map.PLAYER) {
					playerPosition = new Position(i, j);
					grid[i][j] = Map.EMPTY;
				}
			}
		}

		assert playerPosition != null;
		String playerCommand = br.readLine();

		String[] monsterInfos = new String[K];
		for (int i = 0; i < K; ++i) {
			monsterInfos[i] = br.readLine();
		}
		Monster[][] monsters = parseMonsters(N, M, grid, monsterInfos);

		String[] itemInfos = new String[L];
		for (int i = 0; i < L; ++i) {
			itemInfos[i] = br.readLine();
		}
		Item[][] items = parseItems(N, M, grid, itemInfos);

		this.player = new Player(playerPosition, playerCommand);
		this.map = new Map(N, M, grid, player, monsters, items);
		br.close();
	}

	private static Monster[][] parseMonsters(int n, int m, char[][] grid, String[] monsterInfos) {
		StringTokenizer st;
		Monster[][] monster = new Monster[n][m];
		for (String monsterInfo: monsterInfos) {
			st = new StringTokenizer(monsterInfo);
			int R = Integer.parseInt(st.nextToken()) - 1;
			int C = Integer.parseInt(st.nextToken()) - 1;
			String S = st.nextToken();
			int W = Integer.parseInt(st.nextToken());
			int A = Integer.parseInt(st.nextToken());
			int H = Integer.parseInt(st.nextToken());
			int E = Integer.parseInt(st.nextToken());
			monster[R][C] = new Monster(S, W, A, H, E, grid[R][C] == Map.BOSS_MONSTER);
		}
		return monster;
	}

	private static Item[][] parseItems(int n, int m, char[][] grid, String[] itemInfos) {
		StringTokenizer st;
		Item[][] item = new Item[n][m];
		for (String itemInfo: itemInfos) {
			st = new StringTokenizer(itemInfo);
			int R = Integer.parseInt(st.nextToken()) - 1;
			int C = Integer.parseInt(st.nextToken()) - 1;
			char T = st.nextToken().charAt(0);
			if (T == 'W') {
				int attack = Integer.parseInt(st.nextToken());
				item[R][C] = new Item.Weapon(attack);
			} else if (T == 'A') {
				int defense = Integer.parseInt(st.nextToken());
				item[R][C] = new Item.Armor(defense);
			} else {
				String code = st.nextToken();
				item[R][C] = new Item.Accessory(code);
			}
		}
		return item;
	}

	@Override
	public String toString() {
		return map.toString() + "\n"
			+ String.format(Messages.TURN_PRINT_FORMAT, passedTurns) + "\n"
			+ player.toString() + "\n"
			+ reason;
	}
}

class Map {

	public static final char EMPTY = '.';
	public static final char SPIKE_TRAP = '^';
	public static final char MONSTER = '&';
	public static final char BOSS_MONSTER = 'M';
	public static final char WALL = '#';
	public static final char PLAYER = '@';
	public static final char ITEM_BOX = 'B';

	private final int n, m;
	private final char[][] grid;
	private final Monster[][] monsterInfo;
	private final Item[][] itemInfo;
	private final Player player;

	Map(int n, int m, char[][] grid, Player player, Monster[][] monsterInfo, Item[][] itemInfo) {
		this.n = n;
		this.m = m;
		this.grid = grid;
		this.player = player;
		this.monsterInfo = monsterInfo;
		this.itemInfo = itemInfo;
	}

	Monster getMonster(Position position) {
		return monsterInfo[position.r][position.c];
	}

	void removeMonster(Position position) {
		grid[position.r][position.c] = Map.EMPTY;
		monsterInfo[position.r][position.c] = null;
	}

	Item getItem(Position position) {
		return itemInfo[position.r][position.c];
	}

	void removeItem(Position position) {
		grid[position.r][position.c] = Map.EMPTY;
		itemInfo[position.r][position.c] = null;
	}

	char getValue(Position position) {
		return grid[position.r][position.c];
	}

	boolean isWall(Position position) {
		return grid[position.r][position.c] == Map.WALL;
	}

	boolean isValid(Position position) {
		return 0 <= position.r && position.r < n && 0 <= position.c && position.c < m;
	}

	@Override
	public String toString() {
		StringBuilder sb = new StringBuilder();
		for (int i = 0; i < n; ++i) {
			for (int j = 0; j < m; ++j) {
				sb.append((player.isAlive() && i == player.position.r && j == player.position.c) ? Map.PLAYER : grid[i][j]);
			}
			sb.append("\n");
		}
		sb.setLength(sb.length() - 1);
		return sb.toString();
	}
}

class Player {

	Position position;
	final Position initialPosition;
	int level, hp, attack, defense, exp;
	int maxHp;
	Item.Weapon weapon;
	Item.Armor armor;
	int accessoryCount;
	Item.Accessory[] accessories = new Item.Accessory[4];
	boolean activateReincarnation;
	char[] command;
	int commandIndex, commandLength;

	Player(Position position, String command) {
		this.position = position.deepCopy();
		this.initialPosition = position.deepCopy();
		this.maxHp = this.hp = 20;
		this.level = 1;
		this.attack = 2;
		this.defense = 2;
		this.exp = 0;
		this.commandIndex = 0;
		setCommand(command);
	}

	Position nextPosition(char direction) {
		int nr = this.position.r;
		int nc = this.position.c;
		if (direction == 'L') --nc;
		else if (direction == 'R') ++nc;
		else if (direction == 'U') --nr;
		else if (direction == 'D') ++nr;
		return new Position(nr, nc);
	}

	void move(Position position) {
		this.position = position.deepCopy();
	}

	void reincarnation() {
		hillFullHp();
		this.position = this.initialPosition.deepCopy();
		this.activateReincarnation = false;
	}

	boolean isAlive() {
		return this.hp > 0;
	}

	void increaseHp(int hp) {
		this.hp = Math.min(this.maxHp, this.hp + hp);
	}

	void hillFullHp() {
		this.hp = this.maxHp;
	}

	/**
	 * if true, player die.
	 */
	boolean decreaseHp(int hp) {
		this.hp = Math.max(0, this.hp - hp);
		if (this.hp == 0 && hasAccessory(Item.Accessory.Effect.REINCARNATION)) {
			removeAccessory(Item.Accessory.Effect.REINCARNATION);
			this.activateReincarnation = true;
		}
		return this.hp == 0;
	}

	void stepSpikeTrap() {
		int damaged = 5;
		if (hasAccessory(Item.Accessory.Effect.DEXTERITY)) {
			damaged = 1;
		}
		boolean died = decreaseHp(damaged);
		if (died && !activateReincarnation)
			throw new GameEndException(
				String.format(Messages.KILLED_BY, "SPIKE TRAP")
			);
	}

	void battleWith(Monster monster) {
		boolean firstAttack = hasAccessory(Item.Accessory.Effect.COURAGE);
		boolean firstDamaged = false;
		if (monster.isBoss && hasAccessory(Item.Accessory.Effect.HUNTER)) {
			firstDamaged = true;
			hillFullHp();
		}
		while (this.hp > 0 && monster.hp > 0) {
			int attack = Math.max(1, getAttack(firstAttack) - monster.defense);
			boolean monsterDie = monster.decreaseHp(attack);
			if (monsterDie) {
				increaseExp(monster.exp);
				if (hasAccessory(Item.Accessory.Effect.HP_REGENERATION)) {
					increaseHp(3);
				}
				return;
			}

			int damaged = Math.max(1, monster.attack - getDefense());
			if (firstDamaged) {
				damaged = 0;
				firstDamaged = false;
			}

			boolean died = decreaseHp(damaged);
			if (died) {
				if (this.activateReincarnation) {
					monster.hillFullHp();
					return;
				}
				throw new GameEndException(
					String.format(Messages.KILLED_BY, monster.name)
				);
			}

			if (firstAttack)
				firstAttack = false;
		}
	}

	void equipItem(Item item) {
		if (item.type == Item.Type.WEAPON) {
			weapon = (Item.Weapon) item;
		} else if (item.type == Item.Type.ARMOR) {
			armor = (Item.Armor) item;
		} else if (item.type == Item.Type.ACCESSORY) {
			Item.Accessory accessory = (Item.Accessory) item;
			if (!hasAccessory(accessory.effect) && accessoryCount < 4) {
				accessories[accessoryCount++] = accessory;
			}
		}
	}

	boolean hasAccessory(Item.Accessory.Effect effect) {
		for (int i = 0; i < accessoryCount; ++i) {
			if (accessories[i].effect == effect) {
				return true;
			}
		}
		return false;
	}

	void removeAccessory(Item.Accessory.Effect effect) {
		if (accessoryCount <= 0)
			return;

		for (int i = 0; i < accessoryCount; ++i) {
			if (accessories[i].effect == effect) {
				accessories[i] = accessories[--accessoryCount];
				break;
			}
		}
	}

	void increaseExp(int exp) {
		if (hasAccessory(Item.Accessory.Effect.EXPERIENCE)) {
			exp = (int) (exp * 1.2);
		}
		this.exp += exp;
		if (this.exp >= level * 5) {
			++level;
			this.exp = 0;
			maxHp += 5;
			attack += 2;
			defense += 2;
			hillFullHp();
		}
	}

	int getAttack() {
		return getAttack(false);
	}

	int getAttack(boolean isFirstAttack) {
		int attack = this.attack + (weapon != null ? weapon.attack : 0);
		if (isFirstAttack) {
			if (hasAccessory(Item.Accessory.Effect.COURAGE)) {
				attack *= (hasAccessory(Item.Accessory.Effect.DEXTERITY) ? 3 : 2);
			}
		}
		return attack;
	}

	int getDefense() {
		return this.defense + (armor != null ? armor.defense : 0);
	}

	public void setCommand(String command) {
		this.command = command.toCharArray();
		this.commandLength = command.length();
	}

	@Override
	public String toString() {
		return "LV : " + level
			+ "\nHP : " + hp + "/" + maxHp
			+ "\nATT : " + attack
			+ "+" + (weapon != null ? weapon.attack : "0")
			+ "\nDEF : " + defense
			+ "+" + (armor != null ? armor.defense : "0")
			+ "\nEXP : " + exp + "/" + (level * 5);
	}
}

class Monster {

	String name;
	boolean isBoss;
	int attack, defense, hp, exp;
	int maxHp;

	public Monster(String name, int attack, int defense, int hp, int exp, boolean isBoss) {
		this.name = name;
		this.attack = attack;
		this.defense = defense;
		this.hp = hp;
		this.maxHp = hp;
		this.exp = exp;
		this.isBoss = isBoss;
	}

	/**
	 * if true, monster die.
	 */
	boolean decreaseHp(int hp) {
		this.hp = Math.max(0, this.hp - hp);
		return this.hp == 0;
	}

	void hillFullHp() {
		this.hp = this.maxHp;
	}
}

abstract class Item {

	Type type;

	public enum Type { WEAPON, ARMOR, ACCESSORY }

	public static class Weapon extends Item {
		int attack;

		public Weapon(int attack) {
			this.type = Type.WEAPON;
			this.attack = attack;
		}
	}

	public static class Armor extends Item {
		int defense;

		public Armor(int defense) {
			this.type = Type.ARMOR;
			this.defense = defense;
		}
	}

	public static class Accessory extends Item {
		Effect effect;

		public Accessory(String effectCode) {
			for (Effect effect: Effect.values()) {
				if (effect.code.equals(effectCode)) {
					this.type = Type.ACCESSORY;
					this.effect = effect;
					return;
				}
			}
			throw new IllegalStateException("unknown accessory type.");
		}

		public enum Effect {
			HP_REGENERATION("HR"),
			REINCARNATION("RE"),
			COURAGE("CO"),
			EXPERIENCE("EX"),
			DEXTERITY("DX"),
			HUNTER("HU"),
			CURSED("CU");

			private final String code;

			Effect(String code) {
				this.code = code;
			}
		}
	}
}

		return new Position(this.r, this.c);
```
